# CozyDo 主题工坊

`CozyDo` 是一个面向 `Linux.do`（Discourse）论坛的用户脚本，提供主题切换、可视化调色、配置导入导出与界面细节增强能力。

## 项目定位
- 目标：在不改后端、不装插件的前提下，快速获得可用且统一的论坛主题体验。
- 方式：通过 Tampermonkey 注入样式与交互逻辑，仅在前台页面生效。
- 数据：仅写入浏览器本地存储，不上传用户数据。

## 主要能力
- 多预设主题：内置浅色与深色预设，支持一键切换。
- 自定义编辑：可直接编辑主题变量，颜色支持色板与文本双输入。
- 结构参数：支持全局圆角、阴影、卡片层次、头部毛玻璃等调节。
- 主题库管理：统一管理预设、自定义快照、导入主题。
- 导入导出：支持单主题与主题合集格式，兼容历史格式导入。
- 入口策略：优先注入右上角入口，必要时自动回退悬浮按钮。
- 论坛适配：自动处理前台路由切换，避免在后台管理页面干扰。

## 目录说明
- `linuxdo-theme.user.js`：主脚本（当前文件名保留以兼容历史安装方式，项目名已更新为 CozyDo）。
- `theme-packs/`：示例主题包与模板。
- `tests/`：脚本级单元测试。

## 安装方式
1. 安装 Tampermonkey 扩展。
2. 新建脚本并粘贴 `linuxdo-theme.user.js` 全部内容。
3. 打开 `https://linux.do` 或其子域名，脚本自动生效。

## 使用说明
1. 点击右上角画笔图标（或右下角悬浮按钮）打开面板。
2. 在“全部主题”中快速应用预设或已保存主题。
3. 在“自定义”中微调变量并保存为新主题快照。
4. 在“导入导出”中完成 JSON 配置分享与迁移。
5. 在“设置”中调整结构参数与行为开关。

## 导入导出格式

### 单主题格式（当前）
```json
{
  "schemaVersion": 2,
  "kind": "custom-page-config",
  "custom": {
    "name": "我的主题",
    "basePreset": "claude-dark",
    "tokens": {
      "--primary": "#e9e3d9"
    }
  }
}
```

### 主题合集格式（当前）
```json
{
  "schemaVersion": 1,
  "kind": "theme-collection",
  "collection": {
    "themes": []
  }
}
```

### 兼容导入
- `theme-pack`（历史格式）
- `full-config`（历史格式）

## 常见问题
- 看不到入口按钮：页面头部重渲染时会自动重挂载，也可在设置中开启悬浮按钮。
- 主题未生效：确认脚本已启用，且当前页面不是 `/admin` 后台路径。
- 想恢复初始状态：在“设置”中点击“重置全部配置”。
